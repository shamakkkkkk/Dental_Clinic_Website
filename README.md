1.Описание:

ALLSMILES — это современный адаптивный веб-сайт для стоматологической клиники.
Проект демонстрирует чистый HTML и CSS с использованием BEM, CSS-переменных, Flexbox и Grid.

Проект ориентирован на адаптивный дизайн и поддерживает различные размеры экранов и мобильные устройства.

2.Особенности:

1.Чистый и структурированный HTML5
2.CSS с использованием BEM и переменных
3.Адаптивная верстка через Flexbox и Grid
4.Hover эффекты для кнопок и интерактивных элементов
5.SEO-ориентированные изображения с alt
6.Легко расширяемая структура для добавления новых секций
